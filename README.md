# mcdowells-pos
A mockup of a famous POS system for fun's sake.

Uses Vue, Typescript, PrimeVue, and Pinia.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn dev
```

### Compiles and minifies for production
```
yarn build
```
